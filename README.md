# MConnect

## Project Overview

**MConnect** is a full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application designed to provide a connected digital platform for users through secure authentication, responsive user interaction, and efficient data management.

The project focuses on building a modern web solution with seamless frontend-backend integration, scalable architecture, and user-friendly design.

## Features

* User Authentication and Authorization
* Responsive Frontend using React
* RESTful APIs with Express and Node.js
* MongoDB Database Integration
* Secure Data Handling
* Interactive and Scalable MERN Architecture

## Tech Stack

* **Frontend:** React.js, Bootstrap/CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Version Control:** Git and GitHub

## Project Structure

```bash
mconnect/

```

## Installation

Clone the repository:

```bash
git clone https://github.com/hardik751/mconnect.git
cd mconnect
```

Install dependencies:

### Frontend

```bash
cd admin
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm run dev
```
